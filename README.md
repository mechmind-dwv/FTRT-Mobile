# FTRT Mobile Open Source

Experimental open-source heliophysics monitoring platform.

## Features

- FTRT calculation
- NOAA Kp integration
- Geomagnetic risk classification
- Android APK build support
- Scientific validation pipeline

## Status

Experimental research software.
No operational forecasting guarantees.

## Build

```bash
pip install -r requirements.txt
buildozer android debug
```
